◆ removeRow()

void ap.week16.Number2DArray.removeRow ( int  matrix[][],
int  row 
)

This method removes the row, and moves all rows from (row + 1) to the top. The last row will be replaced with zero.

Example: Initial: [6, 2, 4, 2] [7, 6, 7, 5] [3, 0, 1, 9] [5, 8, 5, 6]

after the call removeRow(mat, 2) [6, 2, 4, 2] [7, 6, 7, 5] [5, 8, 5, 6] [0, 0, 0, 0]

Postconditions: matrix will be altered.

Parameters
matrixint[][]
rowint the row index to be removed
