Detailed Description

DeckOfCards

This class represents a deck of cards. The constructor initializes the deck in order, and shuffle can be called to shuffle the deck. Shuffle is a three-pass shuffle system, iterating through the deck three times. The method distributeToFourHands returns a 2D card array with the four hands in it.

Finally, the sort method will sort the deck based on suit and rank.
