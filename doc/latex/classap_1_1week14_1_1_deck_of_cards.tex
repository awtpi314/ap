\doxysubsection{Detailed Description}
\doxyparagraph*{\mbox{\hyperlink{classap_1_1week14_1_1_deck_of_cards}{Deck\+Of\+Cards}}}

This class represents a deck of cards. The constructor initializes the deck in order, and \mbox{\hyperlink{classap_1_1week14_1_1_deck_of_cards_a9c00057586a2890deb42f78afaaa7b07}{shuffle}} can be called to shuffle the deck. Shuffle is a three-\/pass shuffle system, iterating through the deck three times. The method \mbox{\hyperlink{classap_1_1week14_1_1_deck_of_cards_a956662f2b989138c3039a6b23909f2d5}{distribute\+To\+Four\+Hands}} returns a 2D card array with the four hands in it. 

Finally, the sort method will sort the deck based on suit and rank.
