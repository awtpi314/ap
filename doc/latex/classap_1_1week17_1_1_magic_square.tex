\doxysubsection{Detailed Description}
\doxyparagraph*{\mbox{\hyperlink{classap_1_1week17_1_1_magic_square}{Magic\+Square}}}

This class is a representation of a potential magic square. The array passed to the \mbox{\hyperlink{}{ructor}} must be a square matrix, with the length and height being equal, non-\/zero numbers. If this condition is not satisfied, the constructor will throw an exception.
